# Single Page Resume

A responsive single-page resume website built with Laravel, showcasing personal information, skills, projects, and contact details with a clean and modern user interface.

## Features

- Responsive Design
- Modern User Interface
- Skills & Projects Showcase
- Contact Section
- PDF Resume Generation
- Clean and Maintainable Architecture

## Tech Stack

- Laravel
- PHP
- Blade
- Tailwind CSS
- Docker
- MySQL

## Getting Started

```bash
composer install
```

```bash
cp .env.example .env
```

```bash
docker compose up -d
```

```bash
php artisan migrate
```
